# Changelog

All notable changes to Mahjong Prime will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [2.5.0] - 2026-04-06

### Added
- Turkish language support
- Privacy policy page (Google Play compliance)
- SECURITY.md with vulnerability reporting guidelines
- Modern Play Store description and metadata

### Changed
- Major UI cleanup and modernization
- Updated privacy policy with dark/gold theme styling
- Improved game description for store listings
- Updated build configuration and package ID

### Fixed
- Security hardening and dependency updates
- Build configuration fixes for proper package identification

## [2.4.0] - 2026-03-12

### Changed
- Rebranded from Green Mahjong to Mahjong Prime
- New branding by CerebrAI-VorTX
- Updated all icons and visual assets
- Updated app metadata and manifest files

## [2.3.0] - 2025-12-01

### Added
- Initial fork and adaptation
- Base Mahjong Solitaire gameplay with 6 layouts
- 3 tile themes (Classic, Fruits, High Visibility)
- 9 language translations
- Game statistics and achievement tracking
- Hint and undo functionality
- Offline-first architecture
- HD graphics for phones and tablets
